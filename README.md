# sussusfetch

[susfetch](https://github.com/sleepntsheep/susfetch) but much more sus **with audio**

## Prerequisites

```bash
sudo apt install libmpg123-dev libao-dev
```

## Installation

```bash
make
make install
```

**Note**: This app install to local user, do not use sudo
